# Real-Time Tracking App

A real-time location tracking web application built using Node.js, Express, Socket.IO, Leaflet.js, and OpenStreetMap.

This app allows users to share and track live locations on an interactive map in real time.

---

## Features

- Real-time location updates
- Live user tracking on map
- Interactive map using Leaflet.js
- OpenStreetMap integration
- Multiple users supported
- Automatic marker updates
- User disconnect handling

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/real-time-tracking-app.git
```

### 2. Navigate to project folder

```bash
cd Realtime-Location-Tracker
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the server

```bash
node app.js
```

or

```bash
nodemon app.js
```

---

## Usage

1. Open the application in your browser.
2. Allow location access when prompted.
3. Open the app in multiple tabs/devices to see live tracking.
4. User locations will update automatically on the map.
